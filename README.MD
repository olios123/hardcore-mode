# Hardcore Mode

Are you ready to raise the stakes on your Minecraft server? **HardcoreMode** is not just another plugin â€” it completely transforms the way players **experience death**. Instead of respawning like nothing happened, players are **temporarily or permanently banned when they die**.

This isn't a punishment â€” it's a **feature**. A **challenge**. A new way to keep your server fresh, exciting, and truly hardcore.


## Features

When a player dies:
- They are banned from the server for a configurable amount of time.
- The ban duration can scale based on how many times they've died before.
- Players with higher ranks can receive customized ban lengths.
- Ban penalties can decay over time if a player survives long enough.

For Admins:
- Set default lives for all players
- Adjust ban time by death count or permission rank

- Enable ban level reductions over time
- Customize death effects (lightning, sounds, messages)

## Deployment

The `.jar` file is created using **Artifacts**. In IntellijIDEA, you need to prepare the project (Project Structure) for compilation as a `JAR`. After building, you'll receive a `.jar` file that can be used as a plugin.
